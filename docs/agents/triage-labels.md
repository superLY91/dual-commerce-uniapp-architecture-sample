# Triage Labels

The skills speak in terms of five canonical triage roles. This file maps those roles to the local status strings used in `docs/issues/`.

| Label in mattpocock/skills | Status in local issue | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer needs to evaluate this issue |
| `needs-info` | `needs-info` | Waiting on reporter for more information |
| `ready-for-agent` | `ready-for-agent` | Fully specified, ready for an agent to implement |
| `ready-for-human` | `ready-for-human` | Requires human implementation |
| `wontfix` | `wontfix` | Will not be actioned |

When a skill mentions a triage role, use the corresponding local issue status.
